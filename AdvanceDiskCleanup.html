Advanced Disk Cleanup Launcher

This page provides a command to run your PowerShell Disk Cleanup script.

For security reasons, web browsers cannot directly execute local scripts. Please follow the steps below to run the cleanup.

How to Run Your Disk Cleanup Script:

  1. Click the "Copy Command" button below to copy the PowerShell command.
  2. Open **PowerShell as Administrator**:
    • Search for "PowerShell" in your Windows Start Menu.
    • Right-click on "Windows PowerShell" and select "Run as administrator".
    • Click "Yes" if prompted by User Account Control (UAC).
  3. In the Administrator PowerShell window, **paste the copied command** (Right-click or Ctrl+V).
  4. Press Enter to execute the script.
